
# Contributing Guide

Valor is a structured operational logic system. When contributing:

## Rules
1. Do not alter logic unless explicitly intended.
2. Maintain structural integrity of the top-level folder layout.
3. Follow numbering conventions strictly.
4. Ensure any added rules do not conflict with existing system behavior.

## Pull Requests
- Provide a description of intended behavior change.
- Include evaluation scenarios demonstrating correctness.
- Run the evaluation suite before submitting.

## Forbidden Changes
- Changes that introduce behavioral inconsistency.
- Deviation from established tone or structural patterns.
